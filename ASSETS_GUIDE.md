## 🖼️ How to Prepare Images

**Step 1: Find Good Photos**
- Best coupleshots from phone
- Clear, well-lit photos
- Both happy/smiling
- High resolution is fine (we'll resize)

**Step 2: Resize to Square**
Use free tools:
- **Photopea.com** (free Photoshop online)
  1. Open image
  2. Image > Canvas Size
  3. Set to 280x280px center
  4. Export as JPG

- **Canva.com** (free design tool)
  1. Upload image
  2. Select "Square (1:1)"
  3. Download PNG/JPG

- **TinyPNG.com** (free image optimizer)
  1. Upload image
  2. Download compressed
  3. Smaller file size, same quality

**Step 3: Compress for Web**
- Use **TinyJPG.com** or **TinyPNG.com**
- Reduces file size without visible quality loss
- Faster loading times
- Recommended: Keep under 200KB per image

**Step 4: Save with Correct Names**
- `image1.jpg` (NOT image1.png, NOT photo1.jpg)
- `image2.jpg` (Exact name matters!)
- `cats.gif` (or cats.jpg if not animated)

1. **Use JPG for Photos** - Better compression than PNG
2. **Keep Aspect Ratio** - Square (1:1) looks best for letter section
3. **Good Lighting** - Photos look better in natural light
4. **Happy Moments** - Use photos where you're both smiling
5. **No Watermarks** - Remove logos/watermarks first
6. **Test Music** - Make sure audio plays without glitches
7. **Backup Original** - Keep originals before editing
8. **Mobile Preview** - Test images on phone
9. **Load Time** - Compress to keep website fast

## 🔄 How to Replace Files Later

**To Update Photos Later:**
1. Find new photos
2. Resize to 280x280px square
3. Compress with TinyPNG
4. Save as `image1.jpg` and `image2.jpg` (overwrite old)
5. Refresh browser (Ctrl+F5)
6. Done!

**To Change Music Later:**
1. Find new song
2. If not MP3, convert using:
   - **Online converter**: cloudconvert.com
   - **Audacity** (free): audacityteam.org
3. Save as `background-music.mp3` (overwrite)
4. Refresh browser
5. Done!

## 🆘 Common Issues

| Issue | Solution |
|-------|----------|
| Image shows broken | Wrong file name or location |
| Music doesn't play | Wrong format (use MP3 only) |
| File too large | Use TinyPNG to compress |
| Can't find files | Check `assets/images/` folder exactly |
| JPG vs PNG confusion | Use JPG for photos, PNG for graphics |
| Animated GIF doesn't animate | Upload as .gif not .jpg |
| Music is too loud | Edit audio file volume before uploading |
| Image quality bad | Use original high-quality photos |
